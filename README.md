# TarotWave

TarotWave 是一个纯静态的互动塔罗网页应用。它把摄像头手势识别、桌面点击兜底、塔罗牌面展示和即时牌意解读整合在一个可直接部署的前端项目里。

项目当前包含 22 张大阿卡那牌、单张与三张牌阵、手势抽牌流程、牌面图片、牌意解读弹窗、复制解读，以及适配无摄像头环境的点击操作。

## 功能

- **手势抽牌**：通过 MediaPipe Hands 识别手势。伸出五指进入俯视，右滑摊牌，移动手掌框选，向下挥动或捏合四指抽牌。
- **点击兜底**：不启用摄像头时，可以点击牌堆、点击牌面完成同样流程。
- **单张 / 三张牌阵**：支持单张即时讯息，也支持「过去 / 现在 / 建议」三张牌阵。
- **三张牌逐张选择**：三张模式下可依次选择 3 张牌，再进入解读。
- **完整大阿卡那牌库**：内置 22 张大阿卡那，每张牌包含中文牌名、罗马编号、牌面图片、正位含义、逆位含义和建议语。
- **牌意解读**：根据抽到的牌、正逆位和牌阵位置生成结构化中文解读。
- **结果展示**：抽牌后显示牌面；点击抽出的牌或完成手势确认后显示牌意解读框。
- **复制解读**：一键复制当前解读文本。
- **本地历史数据**：最近解读会保存到浏览器 `localStorage`，不上传服务器。
- **纯静态部署**：无需后端，可部署到 Cloudflare Pages、GitHub Pages、Netlify、Vercel 等静态托管平台。

## 交互流程

### 使用手势

1. 点击左下角「开启手势」并允许摄像头权限。
2. 伸出五指，牌堆切换到俯视状态。
3. 手掌向右滑动，摊开牌组。
4. 左右移动手掌选择牌。
5. 向下挥动或捏合四指抽出牌。
6. 握拳后重新张开五指，显示牌意解读。

### 使用点击

1. 选择「单张」或「三张」。
2. 点击牌堆进入俯视，再点击摊开牌组。
3. 点击想选的牌。三张模式需要选择 3 张。
4. 点击抽出的牌，显示牌意解读。
5. 结果页点击「立即抽牌」会重置并开始下一轮。

也可以直接点击底部「立即抽牌」按钮，快速生成一次抽牌结果。

## 本地运行

这个项目没有构建步骤，直接用静态服务器打开即可。推荐使用 HTTP 服务预览，因为摄像头能力通常需要安全上下文或本地 HTTP 环境。

```bash
python3 -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

也可以直接打开 `index.html` 查看静态界面，但摄像头权限和外部手势识别库加载可能受浏览器限制。

## 部署

项目根目录就是发布目录，部署时不需要运行 build 命令。

Cloudflare Pages 推荐配置：

- Framework preset: `None`
- Build command: 留空
- Build output directory: `/`
- Root directory: 仓库根目录

摄像头手势识别需要 HTTPS；Cloudflare Pages 会默认提供 HTTPS。

## 隐私说明

摄像头画面只在浏览器本地用于手势识别，不会上传到服务器。牌意历史记录保存在本机浏览器的 `localStorage` 中。
